## 🚀 Running the Programs

1. Clone the repository:

```bash
git clone https://github.com/anant-labs/Python-Practice.git
```

2. Open the repository:

```bash
cd Python-Practice
```

3. Run any program:

```bash
python Basics/hello_world.py
```

> Make sure Python 3 is installed on your system.
